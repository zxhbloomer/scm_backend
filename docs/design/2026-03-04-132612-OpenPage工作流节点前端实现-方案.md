# OpenPage工作流节点前端实现方案

## 1. 背景与问题

后端已新增 `OpenPage`（打开前端页面）工作流组件（数据库 `ai_workflow_component.id=20`），但前端工作流设计器在加载含该节点的工作流时报错：

```
graph.addNode 调用失败: Error: Node with name 'openpage' does not exist
WorkflowDesigner.vue:603
```

**根因**：`WorkflowDesigner.vue:586` 将组件名 `OpenPage` 转小写为 `openpage` 作为 X6 shape 名称，但 `registerX6Nodes.js` 中未注册该 shape。

---

## 5. 详细设计

### 5.1 OpenPageNode.vue（新建）

**路径**：`src/components/70_ai/components/workflow/components/nodes/OpenPageNode.vue`

**参考**：完全对标 `AnswerNode.vue`，区别只有：
- 图标用 `el-icon-monitor`（绿色，表示"打开页面"）
- 说明文字用 "打开前端页面" 而非模型名称前缀

```
节点外观：
┌─────────────────────────────────┐
│  [monitor图标] OpenPage名称      │  ← CommonNodeHeader（白底）
├─────────────────────────────────┤
│  [cpu图标]  已选模型名 或 未选择  │  ← model-line（灰色圆角背景）
└─────────────────────────────────┘
```

### 5.2 OpenPageNodeProperty.vue（新建）

**路径**：`src/components/70_ai/components/workflow/components/properties/OpenPageNodeProperty.vue`

**参考**：`AnswerNodeProperty.vue`，区别：
- 无提示词配置区域（提示词已在后端硬编码）
- 有"模型选择"区域
- 有说明文字（告知用户该节点的功能）

结构：
```
- 功能说明段（el-alert info）
- 模型选择（WfLLMSelector）
```

### 5.3 registerX6Nodes.js（修改）

在文件头部新增 import：
```js
import OpenPageNode from '../components/nodes/OpenPageNode.vue'
```

在函数 `registerAllWorkflowNodes()` 结尾（`TempKnowledgeBase` 之后）追加：
```js
register({
  shape: 'openpage',
  width: 220,
  height: 80,
  component: OpenPageNode,
  ports: {
    groups: {
      target: { position: 'left', attrs: { circle: { r: 4, magnet: true, stroke: '#555', strokeWidth: 1, fill: '#fff' } } },
      source: { position: 'right', attrs: { circle: { r: 4, magnet: true, stroke: '#555', strokeWidth: 1, fill: '#fff' } } }
    },
    items: [
      { group: 'target', id: 'left' },
      { group: 'source', id: 'right' }
    ]
  }
})
```

### 5.4 WorkflowPropertyPanel.vue（修改）

4处改动：

**① import（新增1行）**：
```js
import OpenPageNodeProperty from './properties/OpenPageNodeProperty.vue'
```

**② components（新增1行）**：
```js
OpenPageNodeProperty
```

**③ v-else-if（在 TempKnowledgeBase 条件之后新增）**：
```html

  v-else-if="selectedNode.wfComponent.name === 'OpenPage'"

  :wf-node="selectedNode"

```

**④ getIconClass()（新增1条）**：
```js

```
